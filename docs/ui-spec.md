# UI Spec

## Visual direction

- Bright, soft, rounded, calm
- White and off-white surfaces over airy wallpaper gradients
- Medium weight typography with generous padding
- Samsung-style blue accent with restrained saturation
- Soft shadows, not heavy contrast

## Layout rules

- Home screen uses consistent vertical rhythm
- Widget cards feel embedded, not floating wildly
- Dock is visually grounded and slightly elevated
- Edit mode tray reads as a system panel, not a generic modal
- Settings uses large rounded cards and sentence-case titles

## Motion rules

- Drawer rises from bottom with gentle scale and opacity
- Edit mode tray slides up decisively
- Settings feels like a full sheet
- Folder open/close should feel springy but controlled

## Typography rules

- Sentence case everywhere
- Labels favor clarity over density
- App labels are single-line
- Secondary metadata is quiet but legible

## Immediate prototype goals

- High-fidelity home surface
- High-fidelity app drawer shell
- Settings surface that looks and reads like Samsung
- Edit mode tray for wallpaper/widgets/settings/page management
