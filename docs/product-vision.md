# Product Vision

## Goal

Build a launcher that feels indistinguishable from Samsung One UI Home in layout, motion, terminology, settings structure, and default behavior.

## Product rules

1. Favor Samsung parity over launcher power-user features.
2. Copy behavior before adding customization.
3. Keep the clone project isolated from Lawnchair Lite until parity is proven.
4. Treat Samsung naming as product requirements:
   - Home screen layout
   - Home screen grid
   - Apps screen grid
   - Folder grid
   - Apps button
   - Media page
   - App size
   - App labels
   - Widget labels
   - Lock Home screen layout
   - Swipe down for notification panel
5. Search should behave like Samsung Finder, not like a custom command palette.

## Definition of done

The clone is ready when:

- a Samsung user can pick it up without relearning navigation
- settings names and locations match expectations
- default home, drawer, folders, and edit mode feel native to One UI
- motion, spacing, label behavior, and icon sizing read as Samsung immediately

## Non-goals

- shipping Samsung proprietary assets
- reusing Samsung brand names in the product
- preserving every Lawnchair Lite customization feature
