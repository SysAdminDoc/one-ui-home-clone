# Samsung Parity Checklist

## Home Screen

- Separate `Home screen only` and `Home and Apps screens` layout options
- Exact Samsung home grid options for the target One UI version
- Single-line labels with Samsung-style truncation
- Optional apps button in dock
- Media page support
- Lock home screen layout
- Add new apps to Home screen
- Rotate Home screen on supported devices

## Edit Mode

- Long-press empty space enters Samsung-style edit mode
- Bottom tray includes wallpapers/themes, widgets, settings, page controls
- Page add/remove and default home page selection match Samsung behavior
- Drag/drop and page rearrange feel native

## App Drawer

- Samsung-style drawer open/close motion
- Samsung sort and page behavior for chosen One UI target
- Search/Finder grouping behavior
- Section headers and ordering only if target version uses them
- Settings access from drawer

## Folders

- Folder open animation and sheet appearance match Samsung
- Folder grid options match Samsung settings
- Folder title/header behavior matches Samsung
- Folder edit affordances and close gestures match Samsung

## Widgets

- Widget picker layout and sections feel Samsung-like
- Resize handles, padding, and placement behavior match Samsung
- Widget labels and spacing match target version

## Search

- Finder-style grouped results
- Recent searches
- Settings results
- Contacts and web actions only where Samsung-like
- Remove non-Samsung power-user extras from default experience

## Settings

- Samsung terminology
- Samsung section ordering
- Samsung defaults
- Home screen, Apps screen, Folder, Badge, Search, Gesture, and Advanced areas shaped like One UI

## Motion And Feel

- Samsung easing and spring feel
- Drawer and folder surfaces rise with Samsung-like depth
- Widget and folder shadows match One UI softness
- Touch targets and spacing feel dense but calm

## Large Screens

- Tablet portrait and landscape behavior
- Separate layout handling where Samsung supports it
- Large-screen settings width and structure
