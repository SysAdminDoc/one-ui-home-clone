# Clone Brief

## Working target

Aim for One UI Home parity, not “Lawnchair with Samsung colors.”

## Core experience to copy

- home layout and spacing
- edit mode flow
- settings structure
- drawer and Finder feel
- folder presentation
- media page behavior

## What to avoid

- cloning Samsung proprietary assets
- preserving custom launcher features that weaken parity
- mixing prototype code into Lawnchair Lite before the clone direction is stable
